Privacy Policy

Last updated: 1 May 2026

Since When?! is a Chrome extension that adds creator information to Google Calendar event popups. This policy explains what data it accesses and what happens with it.

What we access

When you click a calendar event, Since When?! makes a request to the Google Calendar API to retrieve the event's creator name, email address, and creation timestamp. This requires read-only access to your Google Calendar data, granted via Google OAuth.

What we don't do

  • We do not store any calendar data
  • We do not send data to any server other than Google's own Calendar API
  • We do not track usage, clicks, or behaviour
  • We do not have a backend — the extension runs entirely in your browser

Local caching

Event creator details are cached in memory for up to 5 minutes to avoid repeated API calls during a session. This cache is cleared when the browser tab is closed. Nothing is written to disk or synced.

OAuth token

Your Google OAuth token is managed by Chrome's built-in identity API. It is not stored or accessible by Since When?! directly — Chrome handles token storage and refresh.
